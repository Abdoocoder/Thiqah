# Product

## Register

product

## Users

Store owner or manager at "الثقة للمنتجات البلدية" (Al-Thiqah Local Products), a Jordanian heritage food brand selling olive oil, cheese, and ghee. The user manages the online store — products, orders, customer inquiries, and promotional offers — as their primary daily workflow. Arabic-first, desktop-primary, occasionally on tablet.

## Product Purpose

Admin dashboard for managing an e-commerce operation: CRUD for products and offers, order lifecycle tracking, customer contact inbox, and user role management. Success means the owner can run the entire online store without technical support.

## Brand Personality

Trusted, Local, Premium. Warm earth tones (olive green), heritage-rooted but modern. Professional without being corporate. Approachable without being casual.

## Anti-references

No specific anti-references. The bar is a clean, functional admin panel that feels intentional — not a generic template. Avoid cluttered dashboards, excessive cards, and aimless decoration.

## Design Principles

- **Clarity over cleverness**: The interface is a tool first. Every element has a job. Don't decorate what should be functional.
- **Arabic-first rhythm**: RTL is the default, not an afterthought. Typography and spacing respect Arabic script anatomy.
- **Consistent quality**: Every page, state, and interaction gets the same attention. No rough edges.
- **Respect the user's time**: Fast interactions, clear feedback, minimal friction for frequent actions.

## Accessibility & Inclusion

Target WCAG AA. Adequate contrast ratios, visible focus indicators for keyboard navigation, `prefers-reduced-motion` support. The admin panel is internal-use only but should not exclude users with disabilities.
